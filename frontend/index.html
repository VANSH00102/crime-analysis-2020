Project Highlights

What this project covers and what it demonstrates

🧹

Data Preprocessing

Handles missing values, type coercion, outlier detection, and derived metric calculation using Pandas.

📈

Statistical Analysis

Crime distribution, state rankings, conviction rates, and chargesheet efficiencies across 28+ states.

🗺️

Rich Visualizations

8 professional charts — bar, pie, heatmap, scatter, grouped bar, and stacked composition charts.

🔌

REST API Backend

Flask serves 5 API endpoints for data, stats, images, and on-demand analysis with CORS enabled.

💡

Dynamic Frontend

Vanilla JS fetches live data, renders tables, loads charts, and handles errors gracefully.

📋

NCRB Data Structure

Realistic dataset modelled on NCRB 2020 — covering 28 states, 10 crime types, 280 records.
